What are different types of applications in java?
There are four types of application : 
1. standalone / desktop / window application : J2SE : .jar files
2.Web application		:Servlet , JSP  : Struts(Servlet), JSF(Build on top of JSP) : .war files
3.Enterprise application : EJB : Spring : .ear file
4.Distributed application : Web Serices(SOAP-->XML REST - Json) : .aar files

.jar : java archive file
.war : web archive file
.ear : enterprise archive file
.aar file : application archive file
